# Personal Portfolio Website (React + Vite)

## 📌 Project Description
This is my personal portfolio website built using React and Vite. It serves as a professional showcase of my design work, projects, and skills as a digital media student with a focus on UI/UX and graphic design. The site is designed to be responsive, modern, and easy to navigate.

---

## ✨ Features

- Multi-page layout using React Router (Home, About, Projects, Contact)
- Responsive design for desktop, tablet, and mobile
- Interactive project showcase with images and live project links
- Contact form with basic validation
- Social media links (GitHub and LinkedIn)
- Styled hero section with background image and overlay
- Clean and modern UI design

---

## ⚙️ Setup Instructions

To run this project locally:

### 1. Install dependencies
```bash
npm install

```
### 2. Start development server
  npm run dev

### 3. Build for production
  npm run build

### 4. Deploy to GitHub Pages
  npm run deploy
